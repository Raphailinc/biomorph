# Biomorph Playground

Интерактивная «эволюция» биоморфов: кликайте по понравившемуся образцу, остальные мутируют от него. Геном можно копировать и загружать вручную.

## Запуск
Просто откройте `index.html` в браузере. Никаких зависимостей.

## Как пользоваться
- Клик по биоморфу делает его родителем — все остальные обновляются с мутациями.
- Кнопка «Новая популяция» генерирует случайный набор.
- «Скопировать геном» — копирует JSON текущего выбранного.
- Вставьте JSON в поле и нажмите «Загрузить в выбранный», чтобы воспроизвести биоморф.

## Файлы
- `index.html` — разметка/контролы.
- `style.css` — оформление (тёмная тема, сетка).
- `script.js` — класс `Biomorph`, отрисовка и логика мутаций/выбора.

Алгоритм: геном — массив пар [команда, значение]. При клике создаётся поколение с одной точкой мутации (иногда дублирование/вставка/удаление), что даёт разнообразие форм. Отрисовка — набор линий с симметрией.
